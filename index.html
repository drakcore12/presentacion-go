Tipos de Datos Primitivos

package main
    import ("fmt")
    
    func main() {
        var a bool = true    // Boolean
        var b int = 5        // Integer
        var c float32 = 3.14 // Floating point number
        var d string = "Hi!" // String
    
        fmt.Println("Boolean: ", a)
        fmt.Println("Integer: ", b)
        fmt.Println("Float:   ", c)
        fmt.Println("String:  ", d)
    }
    
Boolean:  true
    Integer:  5
    Float:    3.14
    String:   Hi!

Variables y Constantes

    package main

Structs y Métodos

Go permite crear tipos personalizados y asociarles métodos:

type Persona struct {
        Nombre string
        Edad   int
    }

    func (p Persona) Saludar() {
        fmt.Println("Hola, soy", p.Nombre)
    }
Hola, soy Miguel

Interfaces y Polimorfismo

Go usa interfaces implícitamente para definir comportamientos:

type Animal interface {
        HacerSonido()
    }

    type Perro struct{}

    func (p Perro) HacerSonido() {
        fmt.Println("Guau")
    }

    func main() {
        var a Animal = Perro{}
        a.HacerSonido()
    }
Guau

Manejo de Errores

Go evita excepciones. Usa valores error:

res, err := dividir(10, 0)
    if err != nil {
        fmt.Println("Error:", err)
    }

    }()
