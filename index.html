    import "fmt"

Tipos de Datos Primitivos

package main
    import ("fmt")
    
    func main() {
        var a bool = true    // Boolean
        var b int = 5        // Integer
        var c float32 = 3.14 // Floating point number
        var d string = "Hi!" // String
    
        fmt.Println("Boolean: ", a)
        fmt.Println("Integer: ", b)
        fmt.Println("Float:   ", c)
        fmt.Println("String:  ", d)
    }
    
Boolean:  true
    Integer:  5
    Float:    3.14
    String:   Hi!

Variables y Constantes

    package main
    import "fmt"
    
    const PI = 3.14
    
    func main() {
        var edad int = 30
        nombre := "Miguel"
    
        fmt.Println("Edad:", edad)
        fmt.Println("Nombre:", nombre)
        fmt.Println("PI:", PI)
    }
    Edad: 30
    Nombre: Miguel
    PI: 3.14

Funciones en Go

Ejemplo:

func saludar(nombre string) string {
        return "Hola " + nombre
    }

Admite múltiples retornos y funciones anónimas.

Condicionales y Switch

  • if, else, else if
  • switch sin break
  • defer: ejecuta al final de la función
if edad >= 18 {
        fmt.Println("Mayor de edad")
    }

Bucles en Go

  • Solo existe for
  • For clásico, tipo while e infinito
  • for range para iterar slices y mapas
for i := 0; i < 5; i++ {
        fmt.Println(i)
    }

Arreglos, Slices y Mapas

  • Array: [5]int — tamaño fijo
  • Slice: []int — dinámico
  • Mapa: map[string]int — clave-valor

Structs y Métodos

type Persona struct {
        Nombre string

    }
func (p Persona) Saludar() {

Manejo de Errores

  • No hay try/catch
  • Se usa error como segundo valor
  • if err != nil patrón común
  • panic para errores fatales
